Aprendendo HTML

Sobre o HTML

HTML, ou Linguagem de Marcação de Hipertexto, é a base da web. É a linguagem usada para estruturar o conteúdo de páginas da internet, definindo como o texto, imagens, e outros elementos são organizados e exibidos. O HTML não é uma linguagem de programação, mas sim uma linguagem de marcação, que usa tags para definir a estrutura e o significado do conteúdo.

Primeiras Tags

Antes de tudo, o que são tags?

Tags são utilizadas para estruturar o html, elas são utilizadas para informar o navegador que tipo de conteúdo está sendo apresentado e em que formato se deseja. As tags geralmente vêm em pares, com uma tag de abertura e uma de fechamento, como "<p> e </p>" para parágrafos, mas algumas tags são auto fecháveis.

Tags de formatação de texto

  • Para o texto em negrito temos duas opções:
      Texto em negrito, só para destacar um texto, sem peso no corpo, use: <b>
      Texto em negrito, mas com peso semântico, com importância no corpo da página, use: <strong>
  • Texto em itálico, use: <i>
  • Texto sublinhado, use: <u>
  • Texto com marcação, tipo marca texto, use: <mark>
  • Para colocar um número ao quadrado 102, coloque o número a ser elevado entre a tag: <sup>
  • Para colorir um texto, usar o atributo da tag utilizada,"style", assim: <p style = "color: blue;"
  • Para evidenciar a correção de um texto,como, R$100 ,utilizamos <del>
  • Para definir uma sigla e mostrar o seu significado ao passar o mouse, dessa forma: HTML utilizamos a tag : <abbr>
  • Temos a tag <font> que serve para definir fonte, cor, tamanho.. porém é uma tag obsoleta pois fazemos isso com o css, veja o exemplo do que ela faz: Texto Formatado

Inserir Imagem:

Para adicionar uma imagem utilizamos a tag <img>, nela existe o atributo "src", é nesse qe iremos colocar o endereço da imagem, que pode ser tanto local como externo.
